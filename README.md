# Our online courses

Интерактивный каталог онлайн-курсов: фильтрация по категориям, поиск по названию и постраничная подгрузка карточек.

## Возможности

- Отображение карточек курсов из локального JSON
- Фильтрация по категориям с подсчётом количества
- Поиск по названию курса
- Подгрузка карточек кнопкой «Load more»
- Адаптивная вёрстка (mobile-first)

## Установка

```bash
git clone <repository-url>
cd test-catalog
npm install
```

## Запуск

### Разработка

Скомпилировать стили и запустить локальный сервер:

```bash
npm run sass:build
npm run dev
```

Откройте в браузере адрес, который выведет `serve` (обычно [http://localhost:3000](http://localhost:3000)).

## Скрипты

| Команда | Описание |
| --- | --- |
| `npm run dev` | Локальный dev-сервер |
| `npm run sass` | Компиляция SCSS в watch-режиме |
| `npm run sass:build` | Однократная сборка стилей |

## Структура проекта

```
test-catalog/
├── data/
│   └── catalog.json      # Данные каталога
├── js/
│   ├── api.js            # Загрузка JSON
│   ├── state.js          # Состояние и фильтрация
│   ├── render.js         # Рендеринг DOM
│   └── main.js           # Точка входа
├── scss/                 # Исходники стилей
├── css/                  # Скомпилированные стили
├── images/               # Изображения курсов и иконки
└── index.html
```

## Стек

- HTML, CSS (SCSS), JavaScript (ES modules)
- [Sass](https://sass-lang.com/) — препроцессор стилей
- [serve](https://github.com/vercel/serve) — статический dev-сервер

## Планы по доработке

- **Debounce для поиска** — откладывать фильтрацию на 150–300 мс, чтобы не перерисовывать список на каждый символ
- **Поиск по авторам** — расширить фильтрацию: искать не только по названию курса, но и по имени автора
